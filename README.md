# CpGFinder

CpGFinder is a gene-level CpG probe identification tool based on Illumina EPIC methylation arrays and UCSC genomic coordinates. It provides an interactive Shiny application for exploring CpG probes associated with genes.

---

## Installation and Run

```r
install.packages("devtools")
devtools::install_github("ytwangZero/CpGFinder")

CpGFinder::run_app()
